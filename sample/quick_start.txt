Quick Start Guide for 'Chimera25_FTR_C_EN_XG-NR_20_4K_20150608_OV' test package:

    Please see README.md for a full documentation.

1. Install external tools needed for conversion:

    1.1 JDK-8 must be used. JAVA_HOME must be set and 'java' must be in the PATH. 

    1.2 Get FFMPEG (https://ffmpeg.org/)

    1.3. Get x264 Encoder (http://www.videolan.org/developers/x264.html).
         10-bit version must be used!
         One can download official builds of x264: [x264-builds](http://download.videolan.org/pub/x264/binaries/)

    1.4. Get BMX: bmx2raw and raw2bmx applications (https://sourceforge.net/projects/bmxlib/).

    1.5. Get ASDCP tool. Use the explicitly provided one which is enhanced to work properly with TTML wrapped in MXF.
         There is a Windows distribution there which can be used out of the box on Windows.

2. Download the source code of IMF Conversion Utility.

3. Build the source code (JDK-8 must be used!):
    ./gradlew clean build
    
4. Go to /build directory to find a distribution.

5. Extract the distribution and go to 'imf-conversion-utility-1.0-alpha' folder. 
   
6. Edit config.xml:

    6.1 Set external executables
       If all the tools above are added to the PATH, then use the following configuration:
        <externalTools>
            <tool id="ffmpeg">ffmpeg</tool>
            <tool id="ffprobe">ffprobe</tool>
            <tool id="bmx">raw2bmx</tool>
            <tool id="mxf2raw">mxf2raw</tool>
            <tool id="x264">x264-10bit</tool>
            <tool id="asdcp-unwrap">as-02-unwrap</tool>
        </externalTools>
    
        If tools are not added to the path, then enter full absolute paths to executables: 
        <externalTools>
            <tool id="ffmpeg">/path/to/ffmpeg</tool>
            <tool id="ffprobe">/path/to/ffprobe</tool>
            <tool id="bmx">/path/to/raw2bmx</tool>
            <tool id="mxf2raw">/path/to/mxf2raw</tool>
            <tool id="x264">/path/to/x264-10bit</tool>
            <tool id="asdcp-unwrap">/path/to/as-02-unwrap</tool>
        </externalTools>
    
    6.2 Enter 'Chimera25_FTR_C_EN_XG-NR_20_4K_20150608_OV' IMF package location (full absolute path):
            <imp>/tmp/TEST/Chimera25_FTR_C_EN_XG-NR_20_4K_20150608_OV/Chimera25_FTR_C_EN_XG-NR_20_4K_20150608_OV</imp>
            
    6.3 Enter the output directory where the flat .mxf file will be generated:
            <workingDirectory>/tmp/imf-conversion-output</workingDirectory>
            
7. Edit metadata.xml if needed (can be used as is for tests).
     
8. Run conversion by executing the following command:
    bin/imf-conversion-utility -f dpp -c config.xml -m convert --metadata metadata.xml --audiomap audiomap.xml
    
9. An output flat file is created under the specified output directory and is called '{output-dir}/output.mxf'.

10. Logs:
    10.1 The current conversion job log file: /logs/imf-utility.log.
    10.2 Previous conversion jobs log files: /logs/archive.
    10.3 External tools log files: {output-dir}/logs.
